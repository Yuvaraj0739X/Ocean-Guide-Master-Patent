# 🌊 Ocean Guide Master — Patented Maritime Safety System

> A compact IoT device protecting Indian fishermen from illegal arrest,  
> maritime border violations, and ocean emergencies.

[![Patent](https://img.shields.io/badge/Patent-ZA202406482B-1D9E75?style=flat)](https://patents.google.com/patent/ZA202406482B/en)

## The Problem
India has 7,600 km of coastline and ranks 3rd globally in fishing population.  
Tamil Nadu fishermen frequently face **illegal arrest by Sri Lankan coast guards**  
due to no real-time border awareness — sometimes detained for months with no evidence of innocence.

## What the Device Does

| Feature | How it works |
|---|---|
| 🗺️ Border Warning | GPS tracks live location, raises alarm when approaching maritime boundary |
| 🚨 Crossing Alert | Sends automatic signal to police + family if border is crossed |
| 📋 Evidence Log | Stores full location history — legal proof if arrested without crossing |
| 🆘 Distress Signal | One-button SOS to emergency responders for health emergencies |

## Hardware Components
`Raspberry Pi` · `Satellite GPS Tracker` · `IMT Module` · `Antenna` · `Battery`

The Raspberry Pi runs Python scripts that continuously compare the fisherman's GPS  
coordinates against stored maritime boundary data and trigger appropriate alerts.
